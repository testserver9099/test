# Swag Tier Images

This directory contains the prize images for each arcade tier.

## Required Images

Please add the following images to this directory:

1. **novice.png** - Arcade Novice tier (25 points)
   - Prize: Arcade Controller (joystick)

2. **trooper.png** - Arcade Trooper tier (45 points)
   - Prize: Premium Arcade Controller (joystick with buttons)

3. **ranger.png** - Arcade Ranger tier (65 points)
   - Prize: Google Cloud Pen + Colorful Pen

4. **champion.png** - Arcade Champion tier (75 points)
   - Prize: Google Cloud Hoodie + Black Pen + Colorful Pen + USB Cable

5. **legend.png** - Arcade Legend tier (95 points)
   - Prize: Google Cloud Hoodie + Black Pen + Colorful Pen + USB Cable

## Image Specifications

- Format: PNG (transparent background recommended)
- Recommended size: 400x400 pixels or larger
- Aspect ratio: Square or 4:3
- Background: Transparent or matching the tier card background

## How to Add Images

1. Save your prize images with the exact filenames listed above
2. Place them in this directory: `public/images/swag/`
3. The images will automatically display in:
   - Arcade Swags Tier section (sidebar)
   - Swag Eligibility section
   - Swag Tier Progress cards

## Fallback Behavior

If an image fails to load or doesn't exist, the UI will automatically show:
- Award icon as fallback
- Gift emoji placeholders for rewards
